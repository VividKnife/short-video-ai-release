  # ShortVideo AI — Releases

  This repository hosts the official releases for **ShortVideo AI**, a desktop app for AI-powered short
  video script and storyboard generation.

  > Source code lives at [VividKnife/short-video-ai](https://github.com/VividKnife/short-video-ai).

  ## Download

  Go to the [Releases](https://github.com/VividKnife/short-video-ai-release/releases) page and download
  the latest version for your platform:

  | Platform | File |
  |----------|------|
  | macOS | `ShortVideo-AI-x.x.x.dmg` |
  | Windows | `ShortVideo-AI-Setup-x.x.x.exe` |

  ## Auto-Update

  The app checks for updates automatically on launch. When a new version is available, you'll be
  prompted to download and install it.

  ## Installation

  **macOS**
  1. Download the `.dmg` file
  2. Open it and drag **ShortVideo AI** to your Applications folder
  4. Open 'terminal' app in your Mac.
  5. copy and past this command  
     ```xattr -cr "/Applications/ShortVideo AI.app"```
  6. Launch from Applications

  **Windows**
  1. Download the `.exe` installer
  2. Run it — the app installs and launches automatically
